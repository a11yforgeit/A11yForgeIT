## Features
- Responsive, mobile-first layout
- Accessibility features: skip link, keyboard-friendly navigation, ARIA landmarks
- Light/dark theme (toggle) and font-size adjustments
- Prebuilt pages: Home, About, Contact, FAQ, Docs, Terms, Privacy, Accessibility
